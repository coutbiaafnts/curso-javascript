### Auto-atribuição

- n = n + 4 -> `n += 4`
- n = n - 4 -> `n -= 4`
- n = n * 4 -> `n *= 4`
- n = n / 4 -> `n /= 4`
- n = n ** 4 -> `n **= 4`
- n = n % 4 -> `n %= 4`

### Incremento/Decremento

- x = x + 1 -> `x++` (pós-incremento)
- x = x - 1 -> `x--` (pós-decremento)
- x = x + 1 -> `++x` (pré-incremento)
- x = x - 1 -> `--x` (pré-decremento)
